# 项目介绍

本项目面向**电子制造 + 整机组装**混合型车间的 MES（制造执行系统）场景，构建面向生产现场的智能 Agent 能力，用于支撑工艺执行、质量追溯、异常协同与决策辅助。

车间采用典型的**两段式生产架构**：前段为 PCBA 电子制造，后段为整机装配与交付。两类产品在物料形态、工艺路径与质量管控点上存在显著差异，但在工单、批次、设备、人员与质量数据层面需实现统一建模与贯通追溯。

---

## 一、电子制造段（PCBA）

电子段以 PCB 为载板，经表面贴装（SMT）与通孔插装（THT）等工艺，产出可供下游装配使用的印制电路板组件（PCBA）。

### 1.1 SMT 表面贴装流程

| 序号 | 工序 | 说明 |
|------|------|------|
| 1 | PCB 来料检验（IQC） | 对基板外观、尺寸、焊盘状态等进行入厂检验 |
| 2 | 镭雕打标 | 在 PCB 表面进行激光雕刻，写入序列号、版本号等追溯标识 |
| 3 | 锡膏印刷 | 通过钢网将焊膏精确印刷至焊盘 |
| 4 | 锡膏检测（SPI，可选） | 检测焊膏厚度、覆盖率与偏移，预防贴装缺陷 |
| 5 | 贴片（SMT） | 高速贴片机完成阻容、IC 等表贴元器件贴装 |
| 6 | 回流焊 | 按温区曲线完成焊膏熔化与焊点成型 |
| 7 | AOI 自动光学检测 | 对焊点、元器件偏移、缺件、立碑等缺陷进行自动识别 |
| 8 | 人工目检 | 对 AOI 存疑区域及关键位进行人工复核 |

### 1.2 THT 通孔插装及后处理流程

SMT 完成后，对仍需通孔安装的元器件执行插装及后续工艺。典型顺序如下（具体以产品 BOM 与工艺路线为准）：

| 序号 | 工序 | 说明 |
|------|------|------|
| 1 | 插件 | 将变压器、接插件、大功率器件等通孔元件插入 PCB |
| 2 | 波峰焊 / 选择性焊接 | 完成通孔引脚焊接；混装板或热敏感器件可采用选择性焊接 |
| 3 | 手工焊接（按需） | 对特殊大件、返修件或无法自动焊接部位进行补焊 |
| 4 | 清洗（按需） | 去除助焊剂残留，满足后续涂覆或测试要求 |
| 5 | 三防涂覆 | 对 PCBA 进行防潮、防霉、防盐雾保护 |
| 6 | 点胶（按需） | 对特定元器件或区域进行加固、密封或绝缘处理 |

### 1.3 PCBA 测试与交付

| 序号 | 工序 | 说明 |
|------|------|------|
| 1 | 工装测试（ICT / FCT） | 通过专用测试夹具进行在线测试或功能测试，验证电气性能 |
| 2 | PCBA 终检与入库 | 完成外观与功能确认后，作为半成品进入装配段或成品库 |

## 二、整机组装段

装配段以 PCBA 为核心，配合结构件、面板、线束、紧固件等物料，经机械组装与软件配置，形成可交付的整机产品。

| 序号 | 工序 | 说明 |
|------|------|------|
| 1 | 领料与齐套 | 按工单领取 PCBA、外壳、面板、辅料等，完成物料齐套校验 |
| 2 | 机械组装 | 通过锁螺丝、压合、贴胶、卡扣装配等方式完成整机结构集成 |
| 3 | 软件烧录 | 写入固件/应用程序，完成设备软件配置 |
| 4 | 整机功能测试 | 在整机工况下验证通信、接口、显示及核心业务功能 |
| 5 | 老化测试 | 通过规定时长与工况运行，筛除早期失效产品 |
| 6 | 终检 | 外观、标识、附件完整性及出厂指标复核 |
| 7 | 包装入库 | 完成标签、说明书、附件配套后入库或发运 |

## 三、项目定位

本 Agent 项目服务于上述**电子 + 装配**一体化车间的 MES 应用建设，重点覆盖：

- **工艺路线建模**：支持 SMT、THT、测试、装配等多段工艺的标准化定义与版本管理
- **生产过程管控**：工单下发、工序报工、设备状态、在制品（WIP）跟踪
- **质量与追溯**：批次/序列号贯穿 PCBA 至整机，关联 AOI、测试、老化等质量数据
- **现场智能协同**：为工艺查询、异常诊断、排产辅助等场景提供 Agent 化交互能力

车间形态归类为**电子制造与整机组装混合型工厂**，兼具高密度 SMT 自动化产线与面向多品种小批量的装配柔性工位，是 MES 系统中典型的复杂制造场景。
