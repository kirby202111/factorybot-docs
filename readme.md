# 项目介绍

本项目面向**电子制造 + 整机组装**混合型车间的 MES（制造执行系统）场景，构建面向生产现场的智能 Agent 能力，用于支撑工艺执行、质量追溯、异常协同与决策辅助。

车间采用典型的**两段式生产架构**：前段为 PCBA 电子制造，后段为整机装配与交付。两类产品在物料形态、工艺路径与质量管控点上存在显著差异，但在工单、批次、设备、人员与质量数据层面需实现统一建模与贯通追溯。

- 详细工艺流程见 [工艺流程.md](./工艺流程.md)，涵盖 SMT、THT、PCBA 测试及整机组装各工序说明。
- 设备与工装点检保养见 [点检保养.md](./点检保养.md)，涵盖电子段与测试段主机设备及关联工装的管理要求。

---

## 项目定位

本 Agent 项目服务于上述**电子 + 装配**一体化车间的 MES 应用建设，重点覆盖：

- **工艺路线建模**：支持 SMT、THT、测试、装配等多段工艺的标准化定义与版本管理
- **生产过程管控**：工单下发、工序报工、设备状态、在制品（WIP）跟踪
- **质量与追溯**：批次/序列号贯穿 PCBA 至整机，关联 AOI、测试、老化等质量数据
- **现场智能协同**：为工艺查询、异常诊断、排产辅助等场景提供 Agent 化交互能力

车间形态归类为**电子制造与整机组装混合型工厂**，兼具高密度 SMT 自动化产线与面向多品种小批量的装配柔性工位，是 MES 系统中典型的复杂制造场景。
