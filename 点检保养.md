# 点检保养

本文档描述车间设备与工装的**点检**（日常/周期性检查）与**保养**（预防性维护）业务，作为 MES 设备管理模块及 Agent 知识库的基础参考。具体项目、周期与判定标准以设备厂商规范及本厂《设备点检保养规程》为准。

---

## 一、业务概述

### 1.1 点检与保养的定义

| 类型 | 定义 | 典型周期 | 目的 |
|------|------|----------|------|
| **点检** | 对设备运行状态、关键参数、外观及安全条件进行例行检查，确认是否满足开工条件 | 班前、每日、每周 | 早发现异常，防止带病生产 |
| **保养** | 按计划对设备执行清洁、润滑、紧固、校准、更换耗材等维护作业 | 每月、每季、每年 | 延长寿命，保持精度与稳定性 |

### 1.2 管理原则

- **人机料法环**：点检保养须关联设备台账、责任岗位、备件耗材、作业标准及环境要求
- **先检后开**：关键设备未完成当班点检或点检不合格，不得投入生产
- **计划驱动**：保养任务由 MES 按周期自动生成，超期未执行须预警并记录
- **全程留痕**：点检结果、保养记录、异常处置、备件更换均纳入设备履历，支持追溯与统计分析
- **工装同等对待**：测试工装、夹具、钢网等生产辅助装置纳入点检保养体系，与主机设备一并管控

### 1.3 与 MES 的集成要点

- 设备台账维护设备型号、工位、所属产线、点检保养方案
- 按周期下发点检/保养工单，支持移动端扫码执行
- 记录实测值、照片、判定结果（合格 / 不合格 / 待观察）
- 不合格项触发异常流程：停机、报修、复检确认后方可复产
- 统计设备 OEE 影响因素中的停机、故障与保养达成率

## 二、电子制造段设备

### 2.1 镭雕机

| 类别 | 项目 | 说明 |
|------|------|------|
| 班前点检 | 激光器状态、抽风/除尘 | 确认指示灯正常，除尘风道畅通 |
| 日点检 | 打标清晰度试打、焦距/镜头清洁 | 试打标准样片，核对字符深度与对比度 |
| 日点检 | 夹具定位、传送轨道 | 检查 PCB 定位销、挡板无松动、无异物 |
| 周点检 | 软件版本、模板参数 | 核对打标模板与当前工单版本一致 |
| 月保养 | 光路清洁、导轨润滑 | 按厂商规程清洁光学器件，润滑运动部件 |
| 季保养 | 激光功率校准 | 校验输出功率，必要时由厂商或计量部门执行 |

### 2.2 锡膏印刷机

| 类别 | 项目 | 说明 |
|------|------|------|
| 班前点检 | 刮刀状态、钢网安装 | 刮刀无缺口、变形；钢网张紧、对位正确 |
| 日点检 | 印刷压力、速度、脱模参数 | 核对与工艺文件一致，试印检查锡膏形态 |
| 日点检 | 锡膏搅拌/回温记录 | 确认锡膏在有效期及回温时长内 |
| 周点检 | 轨道宽度、顶板/支撑 pin | 检查传送平稳性，支撑无磨损偏移 |
| 月保养 | 刮刀更换、传动部位清洁润滑 | 按累计印刷次数或周期更换刮刀 |
| 季保养 | 机械精度校验 | 检查印刷重复精度、平行度 |

**关联工装——钢网**

| 类别 | 项目 | 说明 |
|------|------|------|
| 每次上线 | 钢网外观、开孔堵塞 | 无变形、破损；开孔无锡膏残留堵塞 |
| 周点检 | 钢网张力测试 | 张力低于下限须送修或报废 |
| 月保养 | 钢网清洗 | 超声波或专用清洗，干燥后入库登记 |

### 2.3 锡膏检测机（SPI）

| 类别 | 项目 | 说明 |
|------|------|------|
| 班前点检 | 相机/光源、轨道传送 | 确认成像清晰，传送无卡顿 |
| 日点检 | 标准块/GRR 样片校验 | 用量块或标准板校验测量偏差在允差内 |
| 日点检 | 检测程序版本 | 核对 SPI 程序与 PCB 型号匹配 |
| 周点检 | 镜头清洁、照明均匀性 | 清洁光学部件，排除阴影与过曝 |
| 月保养 | 导轨/皮带润滑、除尘 | 保持运动平稳，防止灰尘影响成像 |
| 季保养 | 精度标定 | 按厂商要求执行全轴精度校准 |

### 2.4 贴片机（SMT）

| 类别 | 项目 | 说明 |
|------|------|------|
| 班前点检 | 气压/真空、供料器状态 | 气源压力达标，真空吸取正常，供料器无卡料 |
| 日点检 | 贴装精度试贴、吸嘴磨损 | 试贴标准板，AOI 或人工复核偏移量 |
| 日点检 | 料站表与 BOM 核对 | 确认料号、站位、极性无误 |
| 周点检 | 丝杆/导轨清洁、原点校准 | 清除异物，执行原点回归与贴装精度自检 |
| 月保养 | 吸嘴清洗更换、润滑 | 按使用次数更换磨损吸嘴，润滑运动轴 |
| 季保养 | 贴装头精度校准、皮带张力调整 | 由设备工程师或厂商执行 |

**关联工装——喂料器、顶针、顶板**

| 类别 | 项目 | 说明 |
|------|------|------|
| 日点检 | 喂料器齿轮/盖板 | 运转顺畅，盖板锁紧 |
| 周点检 | 顶针/顶板磨损 | 检查高度一致性，磨损超限须更换 |
| 月保养 | 喂料器内部清洁 | 清除粉尘与碎料，润滑传动件 |

### 2.5 AOI 检测机

| 类别 | 项目 | 说明 |
|------|------|------|
| 班前点检 | 光源、相机、轨道 | 成像正常，传送平稳 |
| 日点检 | 标准缺陷样片校验 | 用 OK/NG 标准板验证检出率 |
| 日点检 | 检测程序与 PCB 版本 | 程序版本与当前生产型号一致 |
| 周点检 | 镜头/玻璃平台清洁 | 防止灰尘、锡渣造成误判 |
| 月保养 | 运动部件润滑、滤网更换 | 保持伺服/步进运行平稳，散热良好 |
| 季保养 | 光源衰减检测、相机标定 | 亮度不足须更换光源，重新标定坐标 |

## 三、测试段设备与工装

### 3.1 工装测试机（ICT / FCT）

涵盖在线测试（ICT）、功能测试（FCT）等以**测试工装**为核心的测试设备。

**主机设备**

| 类别 | 项目 | 说明 |
|------|------|------|
| 班前点检 | 测试仪自检、通道导通 | 执行设备自检程序，确认测试通道正常 |
| 日点检 | 探针/接口接触电阻 | 抽查关键点位接触电阻在允差内 |
| 日点检 | 测试程序版本 | 核对程序版本号与工单/release 一致 |
| 周点检 | 气压/下压机构、安全光栅 | 气缸压力正常，下压平稳，安全防护有效 |
| 月保养 | 测试仪内部除尘、接口清洁 | 防止积尘导致误测 |
| 季保养 | 测试精度校验、校准件验证 | 使用标准板校验测试系统精度 |

**关联工装——测试夹具 / 针床 / 载板**

| 类别 | 项目 | 说明 |
|------|------|------|
| 每次上线 | 工装外观、定位销、压扣 | 无裂纹、变形；定位准确，压合到位 |
| 日点检 | 探针/顶针磨损、弹簧力 | 探针无弯曲、氧化；弹性适中 |
| 周点检 | 接线与接口松动 | 排线、接插件紧固无破损 |
| 月保养 | 探针清洁、局部更换 | 清洁氧化探针，更换失效探针 |
| 按累计次数 | 探针寿命管理 | 达到设定测试次数后强制更换并登记 |

### 3.2 软件烧录设备

涵盖在线烧录器、编程器、烧录治具及关联软件系统。

| 类别 | 项目 | 说明 |
|------|------|------|
| 班前点检 | 烧录器连接、驱动/软件启动 | 设备联机正常，烧录软件可识别目标芯片 |
| 日点检 | 固件版本、烧录配置文件 | 核对固件包 MD5/版本号与发布单一致 |
| 日点检 | 试烧录与校验 | 抽样烧录并执行 read-back 校验 |
| 周点检 | 线缆/接口磨损、烧录座弹片 | 接口无松动，烧录座弹力正常 |
| 月保养 | 烧录座清洁、线缆整理更换 | 清除氧化，更换老化线缆 |
| 季保养 | 烧录器固件升级、兼容性验证 | 升级后须完成全型号回归试烧 |

**关联工装——烧录治具 / 烧录座**

| 类别 | 项目 | 说明 |
|------|------|------|
| 每次上线 | 治具对位、压合力度 | PCBA 或芯片对位准确，压合无翘曲 |
| 日点检 | 弹片/顶针/contact pin 状态 | 接触良好，无歪斜、断裂 |
| 月保养 | 治具清洁、易损件更换 | 清洁接触面，更换磨损弹片 |

### 3.3 联测设备

涵盖整机或模块级**联合测试工站**（多工序、多接口、多设备协同的综合测试系统）。

| 类别 | 项目 | 说明 |
|------|------|------|
| 班前点检 | 各子系统联机状态 | 确认电源、通信、仪器、机械臂/治具等子系统就绪 |
| 日点检 | 测试序列与限值文件 | 核对测试脚本版本、上下限参数 |
| 日点检 | 标准样机测试（Golden Unit） | 用标准合格机验证测试系统无漂移 |
| 周点检 | 接口线缆、转接板、模拟负载 | 检查磨损、氧化，模拟负载阻值正常 |
| 月保养 | 工控机/服务器磁盘清理、日志归档 | 保障系统稳定，备份测试程序 |
| 季保养 | 仪器校准（万用表、电源、频谱仪等） | 外校或内校，合格证在有效期内 |

**关联工装——联测治具 / 转接板 / 模拟负载**

| 类别 | 项目 | 说明 |
|------|------|------|
| 每次上线 | 治具版本与产品型号匹配 | 防错：型号、版本、条码三重核对 |
| 日点检 | 气动/手动夹具动作、传感器 | 夹紧到位，到位信号可靠 |
| 周点检 | 转接板焊点、接插件 | 无虚焊、松动 |
| 月保养 | 易损件更换（顶针、海绵、缓冲垫） | 按寿命计划更换并记录 |
| 按产品切换 | 换型检查清单 | 换线时执行完整换型点检并签字确认 |

## 四、执行与记录要求

### 4.1 任务生成规则

| 任务类型 | 触发方式 | 示例 |
|----------|----------|------|
| 班前点检 | 每班开工前，按设备清单自动生成 | 贴片机、AOI、镭雕机 |
| 周期点检 | 按日 / 周周期滚动生成 | SPI 标准块校验、钢网张力 |
| 周期保养 | 按固定日历或累计运行时间/次数生成 | 贴片机月保养、探针寿命更换 |
| 异常复检 | 点检不合格或故障修复后手动触发 | 维修完成后复检确认 |

### 4.2 记录字段（MES 最小数据集）

- 设备/工装编号、名称、所属产线工位
- 任务类型（点检 / 保养）、计划时间、实际执行时间
- 执行人、确认人（如需）
- 检查项目、标准值、实测值、判定结果
- 异常描述、处置措施、复检结果
- 更换备件/耗材名称、规格、数量

### 4.3 不合格处置流程

```
点检/保养执行 → 判定不合格 → 设备停机挂牌 → 报修/处置 → 维修或调整 → 复检合格 → 解除停机 → 恢复生产
```

关键测试类设备（SPI、AOI、工装测试、联测）点检不合格时，须同步冻结该工位出站，防止不良品流出。
